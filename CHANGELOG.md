# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-04-15

### Added

- Initial multi-project solution scaffold for the `dotnet-docs` .NET tool.
- System.CommandLine-based CLI with root lookup flow plus `cache path` and `cache clear`.
- Symbol normalization, ranking, Microsoft Learn provider, terminal rendering, JSON rendering, and test coverage for core flows.
